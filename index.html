Bacon.js Tweet

Bacon is the number one source of food, everybody loves it. Now you can show your dedication to bacon on your website as well!

Download this project as a .zip file Download this project as a tar.gz file

1. Include

Simply add bacon.js to your website

2. Baconify

run bacon.init(); from your webconsole or add it to the onload attribute on the body tag, whatever you fancy..

3. Add bacon

Now click the ribbon in the upper right corner in order to add bacon!
